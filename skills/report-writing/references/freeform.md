# Freeform Report Reference

> 通用规则见 [_shared.md](_shared.md)。

适用：`reports/notes/**`、专题/备忘/非标准分析。

## When To Use

- 未指定标准 Student/Class/Major 报告
- 跨粒度或窄窗专题、风险备忘、教学复盘

## Recommended Structure

```markdown
# <标题>
## scope
## key_findings
## actions
## evidence
## limitations
```

可自由增删章节；**必须保留** `## Evidence`、`## Limitations`。

## 原则

发现优先：发现问题 → 收集证据 → 结论 → 行动（勿硬套标准模板）。

## Common Mistakes

- 无证据下结论
- 图表与结论无关
- 引用历史 `reports/` 作证据
