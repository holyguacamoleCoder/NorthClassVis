# Major Report Reference

> 通用规则见 [_shared.md](_shared.md)。

适用：`reports/major/<major>/overview.md`

## Required Sections

| id | 要点 |
|----|------|
| scope | majors、classes、week_range、纳入班数/人数 |
| summary | 专业级核心发现 |
| week_trend | 跨班聚合趋势（非仅 WeekView） |
| question_anchors | 多班共性薄弱项 |
| distribution | 班级间与专业内部分布 |
| actions | 发现→证据→建议 |
| evidence | cite 标签 |
| limitations | 跨班可比性、样本代表性 |

## 本粒度要点

- 专业趋势须来自 **aggregate_data**；WeekView 仅案例辅助（≤6 人）
- 单班现象不得上升为专业结论；须说明聚合方式
- StudentView / PortraitView **禁止** report-chart

## Common Mistakes

- 仅依据 WeekView 下专业趋势结论
- 未说明跨班聚合方式
