# Class2 第 13–15 周班级学情总览

## scope

- **班级**：Class2
- **时间范围**：第 13–15 周
- **覆盖学生数**：86 人（`count_distinct student_ID`）
- **数据源**：`submit_record`（周 13–15）、`week_aggregation`（周 13–15）
- **评分标准**：score 0–3（3 分满分）

## summary

Class2 在第 13–15 周的整体表现呈现 **持续走弱** 态势：

- **总均分 1.46/3.0（约 48.7%）**，距满分尚有较大差距。
- **周均 peak 逐周下降**：0.78 → 0.61 → 0.49，班级整体习题完成效果逐步下滑。
- **薄弱知识点集中**：`r8S3g` 知识点均分仅 0.75（25% 正确率），是全班最大的失分点。
- **班内两极分化严重**：个人均分最高 3.0（满分），最低 0.74，极差达 2.26/3.0，约 1/3 学生均分低于 1.0。

## actions

### 建议一：攻克 `r8S3g` 知识短板

**发现**：`r8S3g` 知识点均分仅 0.75（25% 正确率），远低于其他知识点。  
**证据**：193 次提交，均分 0.75/3.0。[@ref:query-results/599bb1c8df2148bb94439a3fecd73db8.json]  
**建议**：
- 在第 16 周安排一次针对 `r8S3g` 的专题练习或课堂讲评。
- 将该知识点拆解为子知识点，逐项过关。
- 对均分低于 1.0 的学生重点布置该知识点专项训练。

### 建议二：关注低分尾部的 14 名学生

**发现**：约 14 名学生（16%）个人均分低于 1.0，最低仅 0.74。  
**证据**：个人均分极差 2.26/3.0。[@ref:query-results/2a873f2850434208928949366a1b9d9f.json]  
**建议**：
- 对这 14 名学生进行个体诊断，分析是知识层面还是投入度问题。
- 考虑分组帮扶或个别辅导，防止掉队进一步扩大。
- 设置每周最低完成量目标，追踪提交活跃度。

### 建议三：维持班级整体趋势的止跌回升

**发现**：周均 peak_value 从 0.78 连续下降至 0.49，降幅 37%。  
**证据**：week_aggregation 三周 peak 均值。[@ref:query-results/f584eae27bc1440ea2ae6a78df983b14.json]  
**建议**：
- 第 16 周增加一次阶段性回顾测试，帮助班级「回温」。
- 穿插互动环节（如小组对抗、限时闯关），提升学生参与积极性。
- 对 trend 下滑幅度大的学生单独提醒干预。

## limitations

- **时间窗口有限**：本报告仅覆盖第 13–15 周，未包含前半学期的基线数据，无法判断下滑是长期趋势还是短期波动。
- **score 范围局限性**：score 评分 0–3 反映单次答题得分，不能直接等同于知识掌握深度。
- **`peak_value` 含义**：`week_aggregation` 的 peak_value 是综合正确率指标，与 score 均分非同一尺度。
- **无作业类型/题型信息**：当前分析未区分选择题、编程题或简答题，不同类型题目的难度差异可能影响得分。
- **未做纵向跨班对比**：缺少同期其他班级数据，无法判断 Class2 的下滑幅度是否属于普遍现象。
